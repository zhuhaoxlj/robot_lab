## 💡 常见问题 FAQ

### Q1: 训练不收敛怎么办？
**A:** 按以下步骤诊断：
1. 检查观察归一化（是否合理范围）
2. 检查奖励尺度（是否被某项主导）
3. 检查学习率（是否过大或过小）
4. 检查网络架构（是否适合任务）
5. 参考调试工具脚本

### Q2: 如何选择超参数？
**A:** 建议：
1. 从论文推荐的默认值开始
2. 使用较小的学习率（1e-4）
3. 逐步调整一个参数（控制变量）
4. 记录每次实验结果
5. 使用自动化工具（Optuna）

### Q3: 训练需要多长时间？
**A:** 取决于：
- 任务复杂度（Reach: 2-3h, Pick-Place: 8-12h）
- 环境数量（512 环境比 64 环境快 8 倍）
- GPU 性能（RTX 4090 vs RTX 3060）
- 调试次数（预期调试时间 = 训练时间 × 2）

### Q4: 如何平衡实习和学习？
**A:** 建议：
- 70% 时间实践训练和调试
- 20% 时间阅读论文和文档
- 10% 时间总结和记录
- 每天固定时间学习（如上午 9-11 点）
- 下午集中实践（2-6 点）
- 晚上总结复盘（7-8 点）

### Q5: 遇到困难怎么办？
**A:** 寻求帮助的渠道：
1. 查看 Issue Tracker（类似问题）
2. 阅读文档和源码注释
3. 咨询同事或导师
4. 在相关社区提问
5. 记录问题到个人知识库
